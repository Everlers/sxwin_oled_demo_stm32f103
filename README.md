# OLED演示程序（STM32F103C8）
### OLED+LED+IS31FL3737+语音模块

## 演示介绍：
上电OLED显示翔光光电Logo，LED显示板全灭，三秒后进入关机状态。  
关机状态LED显示板的电源图标呼吸闪烁，OLED静态显示电源图标。  
按下开机键后LED显示板的电源图标渐灭然后显示设定温度以及其他状态。OLED将电源图标动画移出后移入开机后的界面。  
LED显示板的速冷指示在开机三秒后熄灭(实际产品或将更长时间)。  
WIFI图标始终保持连接状态(实际产品可能会闪烁或熄灭)。  
为了更好的演示，开机默认打开定时以及除湿和扫风功能。  
语音模块在关机状态下也可唤醒，当讲述指令后程序会先执行开机动作再执行讲述指令。因此可在关机状态下唤醒并打开设备！  

## 操作功能：
K1: 短按电源开关  
K2: 短按定时开关  
K3: 短按扫风开关  
K4: 短按除湿开关  
K5:	短按下调风速  
K6:	短按上调风速  
  
旋钮按下: 模式切换  （制冷模式。制热模式。自动模式。安睡模式。送风模式。）  
旋钮顺时针：  
 1在左下角标签显示“当前温度”或“设定温度”的情况下上调温度  
 2在左下角标签显示“关机时间”的情况下上调时间  
旋钮逆时针：  
 1在左下角标签显示“当前温度”或“设定温度”的情况下下调温度  
 2在左下角标签显示“关机时间”的情况下下调时间  

##### 提示：
当打开定时开关时左下角标签显示“关机时间”时可用旋钮调节定时关机的时间：30分钟~24小时。

## 硬件问题点
因显示板电路问题，IS31FL3737供电必须3.3V  通讯线不能串1K 修改成0R  
所有稳压管接线有误  
蜂鸣器声音较小
